## HulaHop

### Requirements

- PHP >= 7.4
- [composer](https://getcomposer.org/)
- [yarn](https://classic.yarnpkg.com/en/docs/install/#mac-stable)


### How to use it ?

To launch the project for dev purpose, you have to launch these commands : 

```
composer i
symfony server:run
```

```
yarn
yarn run encore dev --watch
```
